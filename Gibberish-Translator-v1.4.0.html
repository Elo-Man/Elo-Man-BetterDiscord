Translating the special code back will result in normal text!

Rules:
If there is a letter after or before A, A-axa
If there is a letter that changes before i, it doesn't change
if letter before nd, becomes ct
n becomes j if there is no letters before it and a letter after it
i replaces other letters if overlap eg. die = io because d and di overlap and cancels
If there is a letter before v, it doesn't change
